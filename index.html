About This Project

The EWU Club Hub was developed as a course project by a team of Computer Science students at East West University. The application aims to streamline event management and club activities for the university community.

Technologies Used

HTML5 CSS3 JavaScript Chart.js Font Awesome Google Fonts

Project Features

  • Event management system for university clubs
  • Club directory with detailed information
  • User registration and authentication
  • Admin dashboard for managing events, venues, sponsors, and vendors
  • Comprehensive reporting and analytics
  • Responsive design for all devices
